## Human Direction Update

Ben's 2026-06-23 direction update is captured in `docs/research_pivot_2026_06_23.md`. The automation should keep the residual-column architecture but shift the scientific emphasis away from chasing tiny CE improvements. CE/perplexity are now guardrails; primary evidence should come from support-width deconfounding, oracle-support regret, functional churn, causal intervention fingerprints, task-free continual-learning retention, finite-update commutator reduction, and dense-teacher residual distillation. After the current residual-capacity decision report, prefer the support-width deconfounding matrix and exhaustive support audit before adding more objective variants.

Ben's 2026-06-27 direction update is captured in `docs/anticipatory_contextual_support_routing.md`. Prioritize anticipatory contextual support routing: predict the nondeployable full-context router's future-context feature vector from causal inputs, then route columns using the predicted features. This supersedes the previous immediate ordering of "synthesize first, then branch" and should be pursued as the next bounded design/pilot path, while retaining the non-CE guardrails from the retention/churn, support-quality, and router/value disentanglement evidence. The branch is not a CE-only promotion path; it must include shuffled predicted-feature, token/position-only, same-student, future-perturbation, and retention/churn controls before any default-router claim.

Ben's 2026-06-28 direction update is captured in `docs/core_periphery_pc_columns_for_acsr.md`. Do not treat the latest ACSR negative-evidence closeout as a reason to abandon columnar residuals. Instead, treat it as evidence that routing innovation alone is insufficient. The next sparse-column hypothesis should combine ACSR-style anticipatory routing among columns with HIBACAML-style core/periphery predictive-coding structure inside each column: task-generic, lower-plasticity core units should be protected and retained, while context-specific peripheral units should be more plastic and preferentially pruned under causal-efficacy tests. Dense/rank/norm and parameter-matched MLP residuals remain mandatory controls, but the next design target should test whether more innovative within-column learning improves retention, commutators, pruning selectivity, intervention fingerprints, and CE guardrails.
